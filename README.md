+ 웹 - HTTP 기반

+ HTTP 메세지에 모든 것을 전송

> HTML, TEXT

> IMAGE, 음성, 영상, 파일

> JSON, XML(API)

> 거의 모든 형태의 데이터 전송 가능

> 서버간에 데이터를 주고 받을 때도 대부분 HTTP 사용


+ 웹 서버(WEB SERVER)

> HTTP 기반으로 동작

> 정적 리소스 제공, 기타 부가기능

> 정적(파일) HTML, CSS, JS, 이미지, 영상

> EX) NGINX, APACHE
